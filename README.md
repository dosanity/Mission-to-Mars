# Mission to Mars

## Project Overview
Web Scraping is a method of gaining data from different sources quickly instead of visiting each website yourself and manually extracting data. The web scraping process is automated using a programming script. Web scraping is a method used by organizations worldwide to extract online data for analysis.

## Resources
+ [NASA's Mars Exploration Program News Feed](https://mars.nasa.gov/news/)

+ [Mars Space Facts](https://space-facts.com/mars/)
